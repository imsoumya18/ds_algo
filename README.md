Quick revision of Data Structures & Algorithms — self-contained C++ files, each with a runnable `main()` example.

Compile and run any file directly, e.g.:

```sh
g++ -std=c++17 "Sorting Algorithms/1_selection_sort.cpp" -o out && ./out
```
